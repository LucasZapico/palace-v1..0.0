### REFERENCES
#### keyboard shortcuts
##### Terminal

Open integrated terminal
```cmd+shift+t```

Close integrated terminal
```cmd+shift+x```

toggle next terminal window
```cmd+shift+k```

toggle focus between terminal and open file
```cmd+opt+f```

##### tree

toggle focus between open file and tree
```ctrl+0```


#### Search
##### Find files
search current tree
```cmd + t```
##### Find Directories
search directories
```cmd + shift + h```
